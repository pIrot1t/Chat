Моя программа чат позволяет:
  1) создать до восьми аккаунтов (я не смог реализовать увеличение массива)
  2) удалять аккаунты
  3) войти в аккаунт или выйти из него
  4) отправлять сообщения в общий чат или конкретному пользователю (не более 255 сообщений на все чаты, по той же причине что и с аккаунтами)
